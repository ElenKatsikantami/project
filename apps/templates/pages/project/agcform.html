Whats is AGS4 ?

AGS4 is a standard for data exchange in GeoTechnical Engineering, and compises of two part:

  1. The text file format for sharing data between companies/projects - for machine 2 machine exchange
  2. And a data dictionary used for the exchange of GeoTechnical data - for GeoTechnical engineer stuff

File Format

The first column are the DATA DESCRIPTORS, and basically describe what the rowâ€™s data means:

  1. GROUP - indicates a new group (spreadsheet tab)
  2. HEADING - the column headings for this group (column in a tab)
  3. UNIT - the units for the columns, is there is one
  4. TYPE - the column type, eg X for text or 2DP for two decimal places (the column formatting)
  5. DATA - one or more data rows conforming to the TYPE (the lines of data after fixed header)
  6. The HEADING, UNIT and TYPE are coolectively called the HEADER and define the column ? (to check)
